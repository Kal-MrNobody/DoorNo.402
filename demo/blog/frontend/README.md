# Cryptology Blog — Frontend Deploy (Vercel)

## Steps

1. Install Vercel CLI if needed: `npm i -g vercel`
2. Update `BLOG_API_URL` in `main.js` to your Render backend URL
3. From this directory, run: `vercel deploy`
4. Follow the prompts (link to your Vercel account, confirm settings)
5. Copy the deployed URL
