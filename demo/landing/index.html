02

Wrap Your Agent's Client

DoorNo.402 acts as a wrapper around your existing HTTP execution client. You don't rewrite your agent's logic. You just wrap its network layer.

Python
TypeScript
from doorno402 import protect, PaymentBlockedError from x402.clients.httpx import x402HttpxClient # 1. Initialize and wrap your client client = protect( x402HttpxClient(account=account), daily_budget=5.00, # Max $5.00 USD rolling 24hrs ) # 2. Your agent uses 'client' normally. # Malicious 402 demands raise errors before signing. try: resp = await client.get("https://api.example.com/data") except PaymentBlockedError as e: print(f"Threat Blocked: {e.result['reason']}")
import { protect, PaymentBlockedError } from "doorno402"; // 1. Wrap the native fetch API const safeFetch = protect(fetch, { dailyBudget: 5.00 }); // 2. Provide safeFetch to your agent try { const resp = await safeFetch("https://api.example.com/data"); } catch (e) { if (e instanceof PaymentBlockedError) { console.log("Threat Blocked:", e.result.reason); } }

The 7-Check Validation Pipeline

Once implemented, every 402 payment response is silently routed through 7 intense validation checks. If any single check fails, the payload is destroyed.

💸

Price Inflation

Cross-references described vs. demanded price.

🌐

ENS Trust Oracle

Scores recipient ENS and on-chain history.

🧠

Prompt Injection

Scans descriptions for LLM system overrides.

📉

Budget Drain

Enforces rolling micro-payment caps.

🔒

TLS Enforcement

Rejects unencrypted HTTP payment requests.

🔄

Redirect Hijack

Ensures domains aren't switched mid-flight.

📦

Fake Delivery

Verifies valid response body post-payment.

04

Execute via KeeperHub

DoorNo.402 validates. KeeperHub executes. By wrapping the KeeperHub MCP client, you create an end-to-end secure execution environment for your agent.

TypeScript
import { interceptAndForward } from "doorno402/mcp"; import { KeeperHubClient } from "@keeperhub/sdk"; // Initialize your KeeperHub execution client const keeperClient = new KeeperHubClient({ apiKey: process.env.KEEPER_KEY }); // Wrap it with DoorNo.402's interceptor const secureAgentClient = interceptAndForward(keeperClient); // The agent uses secureAgentClient to trigger actions. // Clean payloads execute. Malicious payloads drop. await agent.run(secureAgentClient);
