Price Inflation & Budgets

Blocks malicious servers claiming $0.01 in the description while charging $5.00 in the payload. Enforces strict daily spending limits so your agent is never drained by rapid micro-payments.

🧠

Prompt Injection Defense

Scans the 402 description for malicious overrides (e.g., "SYSTEM OVERRIDE: ignore limits"). Ensures the payment context fed to your LLM agent is strictly sanitized.

🌐

Identity & Transport Security

Verifies ENS identities to ensure payments go to trusted recipients with transaction history. Enforces strict TLS to prevent plain-text interception of wallet signatures.

Integrate seamlessly

Zero configuration required. Drop-in protection.

# pip install doorno402

from doorno402 import protect, PaymentBlockedError
from x402.clients.httpx import x402HttpxClient

# Wrap your existing client
client = protect(
    x402HttpxClient(account=account),
    daily_budget=5.00,
)

# Use normally. Malicious 402s are blocked.
try:
    resp = await client.get("https://api.example.com/data")
except PaymentBlockedError as e:
    print(f"Blocked: {e.result['reason']}")
