Step 02 / Core Integration

Wrap your client

You don't need to rewrite your agent's networking logic. Just wrap your existing HTTP or x402 client with the `protect` function.

Python
TypeScript
from doorno402 import protect, PaymentBlockedError from x402.clients.httpx import x402HttpxClient # 1. Wrap your existing client and set limits client = protect( x402HttpxClient(account=account), daily_budget=5.00, # Max $5.00 USD per 24hrs ) # 2. Use normally. If a server demands a malicious 402, # an exception is raised BEFORE the transaction is signed. try: resp = await client.get("https://api.example.com/data") except PaymentBlockedError as e: print(f"Blocked: {e.result['reason']}")
import { protect, PaymentBlockedError } from "doorno402"; // 1. Wrap the native fetch API const safeFetch = protect(fetch, { dailyBudget: 5.00 }); // 2. Use safeFetch everywhere your agent makes requests try { const resp = await safeFetch("https://api.example.com/data"); } catch (e) { if (e instanceof PaymentBlockedError) { console.log("Blocked:", e.result.reason); } }
Step 03 / The Firewall

The 7-Check Pipeline

Once implemented, every 402 response is silently routed through 7 validators. If any fail, the payment is blocked.

💸

Price Inflation

Checks described price vs demanded price.

🌐

ENS Trust

Requires valid ENS & transaction history.

🧠

Prompt Inject

Blocks hidden LLM system overrides.

📉

Budget Drain

Stops rapid micro-payment loops.

🔒

TLS Downgrade

Rejects plain-text endpoints.

🔄

Redirect Hijack

Prevents domain switching mid-flight.

📦

Fake Delivery

Verifies data is returned post-payment.

Step 04 / Execution Layer

KeeperHub Integration

DoorNo.402 handles the security validation, while KeeperHub handles the on-chain execution. Intercept the MCP payload perfectly.

TypeScript
import { interceptAndForward } from "doorno402/mcp"; import { KeeperHubClient } from "@keeperhub/sdk"; // Initialize your KeeperHub execution client const keeperClient = new KeeperHubClient({ apiKey: "..." }); // Wrap it with DoorNo.402 const safeAgentClient = interceptAndForward(keeperClient); // The agent uses safeAgentClient. // Malicious demands are blocked. Clean demands are sent to KeeperHub. await agent.run(safeAgentClient);
